About the website

Automated Sizing Dashboard is a web application where it helps to track or audit the instance types currently set to most customer deployments. Instances would connect to this dashboard to query their designated instance type and if not set properly, they will be resized according to what was defined in the dashboard.

Actions Available

  • Dashboard - View Instances
    • User will be able to view the details of the instances. If in Admin mode, additional functions will be available (edit, delete).
  • Add Instance
    • (Admin mode) users will be able to add a specific instance to set a custom desired instance type. Can be done through single entry or via CSV.
  • Reports
    • A page where you can view and audit all the actions that have been done by a user on each instance. And, provides view to the attempts made by environment restart if resizing is successful or not.
