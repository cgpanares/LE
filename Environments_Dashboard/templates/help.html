About the website

LE Environment Dashboard is a web application where it helps to track or audit changes/actions done in a specific LE TEST Environment. The aim of this project is to have a centralized logging of all actions done for all environments we have for testing and for future environments to be deployed.

Actions Available

  • Dashboard - View Instances
    • User will be able to view the instances of the deployment/alias including if they are occupied or not. This button will only appear if the deployment/alias is occupied.
  • Dashboard - Check In
    • User will be able to vacant the deployment/alias. Usually this is used when the user is done using the deployment/alias for testing.
  • Dashboard - Edit
    • User will be able to edit few information while the deployment/alias is occupied.
  • Dashboard - Check Out
    • User will be able to occupy the chosen deployment/alias.
  • Deployment - Include Deployment
    • User has the capability to include APP deployment/alias in the dashboard. Only add those deployment/alias that are used by the team.
  • Deployment - Exclude Deployment
    • User has the capability to remove the APP deployment/alias in the dashboard. This will make the user not to be able to manage the deployment/alias in the dashboard anymore and it needs to be re-added again. This does not do anything on the deployment/alias itself in the APP side.
  • Reports
    • A page where you can view and audit all the actions that have been done by a user on each deployment/alias.
